## Теоретическая часть
- **Список (`list`)** — реализация динамического массива. Доступ по индексу: O(1). Вставка/удаление в начало или середину: O(n). Добавление в конец: амортизированно O(1).
- **Связный список** — узлы с данными и ссылками. Вставка/удаление в начало: O(1). Доступ по индексу: O(n).
- **Стек (LIFO)** — `push`, `pop`, `peek` за O(1). Реализуется через `list`.
- **Очередь (FIFO)** — `enqueue` и `dequeue`. Эффективно реализуется через `collections.deque` (O(1)).
- **Дек** — двусторонняя очередь. Все операции на концах: O(1). Реализация: `collections.deque`.

```python
def insert_at_start(self, data):
    """Вставка элемента в начало списка. Сложность: O(1)."""
    new_node = Node(data)
    new_node.next = self.head
    self.head = new_node
    if self.tail is None:
        self.tail = new_node
```
**Проверка скобок (стек):**
```python
def is_balanced_brackets(expression: str) -> bool:
    stack = []
    mapping = {')': '(', ']': '[', '}': '{'}
    for char in expression:
        if char in mapping.values():
            stack.append(char)
        elif char in mapping:
            if not stack or stack.pop() != mapping[char]:
                return False
    return not stack
```
**Проверка палиндрома (дек):**
```python
def is_palindrome(sequence: str) -> bool:
    dq = deque(sequence.lower())
    while len(dq) > 1:
        if dq.popleft() != dq.pop():
            return False
    return True
```
